Popular NuGet Packages
Package Name Description
Popular Database-Related Packages
EntityFramework Entity Framework is Microsoft's recommended data access technology for new applications.
MySql.Data A client access library for MySQL
Dapper A high-performance Micro-ORM supporting SQL Server, MySQL, Squlite, SlqCE, Firebird, and more.
Respawn Intelligent checkpoints for database tests
Popular Web UI Packages
Bootstrap The most popular front-end framework for developing responsive, mobile first projects on the web.
jQuery A concise JavaScript library that simplifies HTML document traversal, event handling, and much more.
Text/Parsing Utilities
Newtonsoft.Json A popular high-performance JSON framework for .NET
HtmlAgilityPack This is an agile HTML parser that builds a read/write DOM and supports plain XPATH or XSLT. It is a .NET code library that allows you to parse "out of the web" HTML files.
Testing Tools
NUnit NUnit features a fluent assert syntax, parameterized, generic and theory tests and is user-extensible.
Selenium.WebDriver Selenium is a set of different software tools each with a different approach to supporting browser automation.
Logging
NLog NLog is a logging platform for .NET with rich log routing and management capabilities. NLog supports traditional logging, structured logging and the combination of both.
Apache log4net log4net is a tool to help the programmer output log statements to a variety of output targets.
Design Pattern Libraries
AutoMapper A convention-based object-object mapper.
MediatR Simple, unambitious mediator implementation in .NET
Communication Libraries
Microsoft.AspNet.SignalR Incredibly simple real-time web for .NET. This package pulls in the server components and JavaScript client required to use SignalR in an ASP.NET application.
Microsoft.AspNet.SignalR.Client .NET client for ASP.NET SignalR.
Microsoft.AspNet.SignalR.JS JavaScript client for ASP.NET SignalR.
SSH.NET SSH.NET is a Secure Shell (SSH) library for .NET, optimized for parallelism and with broad framework support.
